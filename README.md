# sekai-gaku
世界学の研究発信を目的としたリポジトリです。
Open Notebook Scienceとして運用するため、掲載される知見は必ずしも正確なものとは限りません。
また、個人で研究しているため編集者も私（kotatako）のみとなります。
研究内容を少しでも社会に還元できることを目指しています。
